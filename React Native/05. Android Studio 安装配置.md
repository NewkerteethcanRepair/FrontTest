## 4. 配置 SDK 环境变量

### 配置 ANDROID_HOME

右键点击「我的电脑」- 属性 - 高级系统设置 - 环境变量 - 新建

![image-20200326222439680](assets/image-20200326222439680.png)

在变量名中输入“ANDROID_HOME”，变量值中输入 SDK 所在路径，然后依次确定即可。
